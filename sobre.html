Sobre mim

Iniciei minha trajetória com o curso Técnico em desenvolvimento de sistemas SENAI Florianópolis (CTAI) e tenho desenvolvido projetos pessoais desde então para praticar e aperfeiçoar minhas habilidades de front-end e back-end.

Algumas das tecnologias com a qual já tenho familiaridade são: uso de git e github para versionamento de código e construção de projetos; HTML, CSS, Javascript, Jquery, React e Angular9 no desenvolvimento front-end; framework Django, Javascript e Python no desenvolvimento back-end usando SQlite e PostgreSQL como ferramentas de gerenciamento de banco de dados.
